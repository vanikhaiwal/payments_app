# Payments App - MERN Stack

A full-stack **MERN** (MongoDB, Express.js, React.js, Node.js) application that allows users to **sign up**, **log in**, and **send dummy money** to other users.

## Features

- **User Authentication**: 
  - Sign up with email and password
  - Login with credentials
  - Store user session with JWT (JSON Web Tokens)

- **Send Money**:
  - Users can send dummy money to other users.
  - Simulate transactions between users.
 
  
## Tech Stack

- **Frontend**: React.js, React Router, Axios
- **Backend**: Node.js, Express.js
- **Database**: MongoDB (or a dummy in-memory database for the current project)
- **Authentication**: JWT (JSON Web Tokens)
